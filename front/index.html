Systema de predicao ao ataque do coração


Este sistema foi desenvolvido para auxiliar no diagnostico de classificação ao ataque do coração.

Para isso, basta preencher os campos abaixo com os dados do paciente e clicar em "Diagnosticar".

Diagnosticar
Name Age Sex Cp Trtbps Chol Fbs Restecg Thalachh Exng Oldpeak Diagnostic

Legendas:

  • Age: age in years
  • Sex(Sex): sex (1 = male; 0 = female)
  • Cp: chest pain type
  • Trtbps: resting blood pressure (in mm Hg)
  • Chol: serum cholestoral in mg/dl
  • Fbs: (fasting blood sugar > 120 mg/dl) (1 = true; 0 = false)
  • Restecg: resting electrocardiographic results Value 0: normal , Value 1: having ST-T wave abnormality (T wave inversions and/or ST elevation or depression of > 0.05 mV)
  • Thalachh: maximum heart rate achieved
  • Exng: Característica, Binária, "Você monitora as calorias que consome diariamente?"
  • Oldpeak: ST depression induced by exercise relative to rest
  • Outcome: Target
